- **[REST API Client Code Generator](https://github.com/christianhelle/apiclientcodegen)**: Visual Studio extension supporting multiple generators (AutoRest, NSwag, OpenAPI Generator, Kiota) to generate REST API clients from OpenAPI specifications.
- **[HTTP File Generator](https://github.com/christianhelle/httpgenerator)**: Generate .http files from OpenAPI specifications for testing APIs directly in your IDE.
- **[openapi2zig](https://github.com/christianhelle/openapi2zig)**: Generate Zig API client code from OpenAPI v2.0 and v3.0 specifications.
- **[SQL Compact Query Analyzer](https://github.com/christianhelle/sqlcequery)**: Query and manipulate SQL CE databases (3.0–4.0) with a fast, lightweight tool.
- **[cURL Request Generator](https://github.com/christianhelle/curlgenerator)**: Generate cURL requests from OpenAPI specifications.
- **[AutoFaker](https://github.com/christianhelle/autofaker)**: Automatically generate fake data to speed up unit test setup/arrange.
- **[JMeter Test Plan Generator](https://github.com/christianhelle/jmetercodegen)**: Generate Apache JMeter test plans from OpenAPI specifications for load testing.
- **[ResW File Code Generator](https://github.com/christianhelle/reswcodegen)**: Generate strongly-typed classes from ResW files (Visual Studio 2015–2022).
- **[Multi Document Reader for OpenAPI.NET](https://github.com/christianhelle/oasreader)**: Merge external references into a single OpenAPI document using the Microsoft OpenAPI toolset.
- **[SQLite Query Analyzer](https://github.com/christianhelle/sqlitequery)**: Modern cross-platform SQLite database management tool built with C++ and Qt.
- **[Azure DevOps CLI](https://github.com/christianhelle/azdocli)**: CLI for Azure DevOps operations to streamline automation and CI/CD workflows.
- **[HTTP File Runner](https://github.com/christianhelle/httprunner)**: Fast single-binary runner for .http files from the command line.
- **[HttpTestGen - .http File Testing Framework](https://github.com/christianhelle/httptestgen)**: Run .http files as unit tests in .NET
